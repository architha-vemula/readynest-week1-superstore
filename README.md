# ReadyNest Week 1 - Superstore Sales Dashboard

## Project Overview
This project analyzes Superstore sales data using Power BI.

## Features
- Total Sales KPI
- Order Count
- Customer Count
- Product Count
- Sales by Category
- Sales by Region
- Sales by State
- Sales Trend Analysis

## Tools Used
- Power BI
- CSV Dataset
- GitHub

## Dataset
Superstore Sales Dataset
